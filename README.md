# Explainable Artificial Intelligence reveals key surgical parameters in robot-assisted and open radical prostatectomy
This repository contains the complete analysis code for the manuscript by Klein et al., submitted to Scientific Reports.

## Contents
Preprocessing pipeline, model training and nested cross-validation, SHAP computation, and performance evaluation scripts.

## Data availability
The underlying de-identified clinical data are not publicly available due to data protection regulations (EU GDPR 2016/679) and institutional ethics requirements. Data may be made available to individual researchers upon reasonable request and subject to a data use agreement. Please contact the corresponding author: christian.klein@ukbonn.de
